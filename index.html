伪元素实现行级元素换行

.inline-element::after{
content: “\A”;
white-space: pre;
}
Unicode 字符是专门代表换行符的：0x000A 。 在 CSS 中，这个字符可以写作 “\000A”， 或简化为 “\A”。这里我们用它来作为 ::after 伪元素的内容。也就是在元素末尾添加了一个换行符的意思。white-space: pre; 的作用是保留元素后面的空白符和换行符

                arr.push(a[i]);
                hash[a[i]]=true;
            }
        }

    }
